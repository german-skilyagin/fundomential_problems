Функция creatingBadTable создает таблицу плохих символов для алгоритма Бойера-Мура. Она принимает подстроку substring и ее длину m. 

Функция создает пустой массив badTable, который будет хранить информацию о самом правом вхождении каждого символа в подстроку. Затем функция проходит по всем символам подстроки и для каждого символа запоминает его самое правое вхождение в подстроку (индекс + 1). 

Например, если подстрока "abc" имеет длину 3, то badTable['a'] будет равно 1, badTable['b'] будет равно 2, а badTable['c'] будет равно 3. 

Функция возвращает заполненный массив badTable.


Сначала функция вызывает функцию creatingBadTable, которая создает таблицу плохих символов для алгоритма. Затем функция проходит по строке string, начиная с индекса 0 и проверяет, совпадает ли символ в текущей позиции с символом в позиции m - j - 1 в подстроке substring, где j - текущий индекс в подстроке. Если символы не совпадают, то функция использует эвристику плохого символа и сдвигает указатель i на максимально возможное расстояние между текущей позицией и самым правым вхождением несовпадающего символа в подстроке. Если все символы подстроки совпали, то функция добавляет индекс начала подстроки в массив результатов entries и продолжает поиск. Если количество найденных подстрок достигло заданного числа num, то функция завершается и возвращает массив результатов. Если подстрока не найдена, то функция возвращает пустой массив.
