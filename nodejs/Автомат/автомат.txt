Эта функция создает таблицу переходов для алгоритма Кнута-Морриса-Пратта. Сначала она определяет алфавит подстроки t, затем создает двумерный массив delta для хранения таблицы переходов. Затем она инициализирует таблицу переходов, заполняя ее значениями, используя алгоритм Кнута-Морриса-Пратта. В результате функция возвращает массив из алфавита и таблицы переходов. 

Первый аргумент функции - подстрока t, а второй - ее длина m. Функция определяет алфавит подстроки t, перебирая все символы подстроки и добавляя их в массив alf. Затем создается двумерный массив delta, в котором будут храниться значения таблицы переходов. Для этого создается массив delta с m+1 строками и пустыми массивами в каждой строке.

Затем функция инициализирует таблицу переходов, заполняя нулями все ячейки. Для этого перебираются все символы алфавита и заполняются нулями ячейки delta[0][i].

Далее функция заполняет таблицу переходов, используя алгоритм Кнута-Морриса-Пратта. Для каждого символа подстроки t функция определяет текущее состояние curr и производит переход вперед для этой буквы, заполняя ячейку delta[j][t.charAt(j)] значением j+1. Затем для каждого символа алфавита i функция определяет следующее состояние delta[j+1][i], используя предыдущее состояние delta[curr][i].

В конце функция возвращает массив из алфавита alf и таблицы переходов delta.



Первые пять аргументов функции - это подстрока t, ее длина m, строка s, ее длина n и количество вхождений num, которое нужно найти. 

Сначала функция вызывает функцию creatingTable(t, m), которая создает таблицу переходов delta для алгоритма Кнута-Морриса-Пратта и возвращает массив из алфавита alf и таблицы delta.

Затем функция начинает перебирать символы строки s по одному. Для каждого символа она определяет, принадлежит ли он алфавиту подстроки t. Если символ не принадлежит алфавиту, то текущее состояние condition сбрасывается в 0. Если символ принадлежит алфавиту, то функция производит переход в новое состояние, используя таблицу delta.

Если новое состояние condition равно m (длине подстроки t), то функция находит индекс начала найденного вхождения подстроки t в строку s и добавляет его в массив entries. Если количество найденных вхождений достигло num, то функция возвращает массив из entries, таблицы delta и алфавита alf.

Если вхождения не были найдены, то функция возвращает массив из entries, таблицы delta и алфавита alf.
